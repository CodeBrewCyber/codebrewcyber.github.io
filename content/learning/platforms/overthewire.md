## Overview

OverTheWire is a classic, community-run collection of wargames played entirely over SSH. It's one of the best free resources for building a solid Linux foundation before moving to more complex platforms. The Bandit wargame is the canonical starting point for beginners; subsequent games (Natas, Leviathan, Narnia, Krypton) progressively cover web exploitation, binary exploitation, and cryptography.

## Details

- **Cost:** Free
- **Skill Level:** Beginner → Intermediate
- **Focus:** Linux basics, privilege escalation, web exploitation, binary exploitation, cryptography
- **Link:** [overthewire.org/wargames](https://overthewire.org/wargames)

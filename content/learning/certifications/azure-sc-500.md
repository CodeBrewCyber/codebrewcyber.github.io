## Overview

**SC-500** earns the *Microsoft Certified: Cloud and AI Security Engineer Associate* credential — "Implementing End-to-End Security Controls for Cloud and AI Workloads." It's the successor to AZ-500 (which retires August 31, 2026) and the first Microsoft security exam to test AI workload security as its own discipline. Domains span identity and access, governance, storage/database/network/compute security, posture management, and monitoring with Microsoft Defender for Cloud, Sentinel, and Security Copilot — the same ground my [Cloud > Labs](/cloud/labs/) walkthroughs cover hands-on.

**Status:** In progress

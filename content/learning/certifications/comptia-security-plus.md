## Overview

Security+ is the most widely recognized entry-level security certification and a common baseline requirement (including DoD 8570/8140 roles). The current exam is **SY0-701**, which covers general security concepts, threats and vulnerabilities, security architecture, security operations, and governance/risk/compliance. It's the natural next step after A+ and a strong foundation before specializing into cloud or blue-team work.

**Status:** Earned
